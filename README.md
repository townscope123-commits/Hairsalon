# Glamour Hair Salon Website

A modern, mobile-responsive hair salon website built with pure HTML, CSS, and JavaScript. No installation required!

## Features

- **All Hair Services**: Cutting, Styling, Coloring, Treatments, and Extensions
- **Mobile Friendly**: Fully responsive design that works on all devices
- **Modern UI**: Beautiful gradient design with smooth animations
- **Navigation**: Sticky navigation with mobile hamburger menu
- **Contact Form**: Functional contact form for appointments
- **Service Cards**: Detailed service descriptions with pricing
- **About Section**: Company information with statistics
- **Social Media Links**: Facebook, Instagram, Twitter integration

## Technologies Used

- **HTML5** - Structure
- **TailwindCSS (CDN)** - Styling framework
- **JavaScript (Vanilla)** - Interactivity
- **Lucide Icons (CDN)** - Icon library

## How to Run

**No installation required!** Simply:

1. Open the `index.html` file in your web browser
2. Or double-click on `index.html` to open it directly

That's it! The website will work immediately without any setup.

## Services Included

1. **Hair Cutting** - Professional haircuts for all hair types
2. **Hair Styling** - Blowouts, updos, and event styling
3. **Hair Coloring** - Highlights, balayage, and full color
4. **Hair Treatments** - Keratin, deep conditioning, and repair
5. **Hair Extensions** - Tape-in, micro-link, and fusion methods

## Customization

You can customize the website by editing the `index.html` file:

- **Colors**: Modify the `tailwind.config` script in the HTML head
- **Content**: Edit text, services, and pricing directly in HTML
- **Styling**: Modify Tailwind classes for design changes
- **JavaScript**: Edit the script section at the bottom for functionality

## Deployment

Since this is a single HTML file, you can deploy it anywhere:

1. **Netlify Drop**: Drag and drop the file to netlify.com
2. **GitHub Pages**: Upload to GitHub and enable Pages
3. **Any Hosting**: Upload to any web hosting service
4. **Local Server**: Use any local server (VS Code Live Server, etc.)
